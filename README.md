# To-Do App

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript. This app helps users organize their daily tasks by adding, completing, and deleting tasks.

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Responsive design
* User-friendly interface
* Data saved using Local Storage (if implemented)

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Use

1. Enter a task in the input field.
2. Click the **Add** button.
3. Mark tasks as completed when finished.
4. Delete tasks you no longer need.
